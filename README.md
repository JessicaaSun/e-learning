## Getting Started

### Local Installation


1. Clone the repo
   ```sh
   git clone https://github.com/cstadservice/istademy-ui
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Deployment

Deployment: https://istademy-ui.vercel.app
